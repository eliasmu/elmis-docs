Setting Your Server

The E-LMIS has been tested in a number of different linux settings. Most commonly, the E-LMIS has been deployed on the cloud Setting. Getting your server up and running in the cloud environment is fairly straight forward process. The following simple step by step process could be used to configure your linux server environment.

  1. Install Tomcat 7 or above

  2. Install Postgres 9 or above

  3. Download the war file from the following URL

  4. Deploy the war file and enjoy

Note

For best result with your configuration, we recommend that you configure your tomcat server behind an apache server. You could configure apache2 to communicate with the tomcat server using the AJP protocol.
