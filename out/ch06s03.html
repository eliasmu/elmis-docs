FTP Based Integrations

E-LMIS can export order files and upload them on to an FTP server. For this feature to be activated you will have to do the following configurations.

Note

The Current version of the application is capable of exporting Order files in CSV file format. However future versions may be able to support other file formats as well.

The first configuration is one that could be done via the E-LMIS UI where you will be able to select which fields need to be included in the CSV file.
