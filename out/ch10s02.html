Optimizing your server

The following tips could be used to optimize the performance of your E-LMIS configuraiton.

Setup Apache to put an expires header on HTML, CSS and Javascript files. This tells the Browser to use it's own cache for a majority of the static files in the application. This setting reduces the size of files that have to be transported to the client machine every time a page is opened.

Add a deflate header in your apache configuration for JSON files to be comressed using GZIP.
